# Personal Portfolio Website

A simple and responsive personal portfolio website created using HTML and CSS.  
This portfolio showcases my education, projects, technical skills, achievements, and interests.

---

## Features

- Responsive navigation bar
- About Me section
- Education details
- Projects showcase
- Tech Stack section
- Achievements & interests
- Contact/Footer section
- Clean and beginner-friendly design

---

## Technologies Used

- HTML5
- CSS3
